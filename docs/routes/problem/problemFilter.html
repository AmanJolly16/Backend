Problem Filter

Get List of Problems based on various Filter Provided by User.

URL

GET
/problems/

Headers

Header Name Required Type Description
Authorization false string Bearer {Token}

URL Parameters

Param Name Required Type Description
page false integer page number to go to, default 1
per_page false integer number of problems in a page, default 20
tags false string comma separated topic tags
and_in_tags false boolean if true, filter tags by and i.e. all given tags must be present in problem
platform false string comma separated platform - F,C,A,S,U
difficulty false string comma separated difficulty - B,E,M,H,S,C
range_l false integer filter problems rating greater than range_l
range_r false integer filter problems rating less than range_r
index false string comma separated index, to search for index example a,b,c,d,e
search false string comma separated search key, to search in problem id, name, url, tags, contest id
mentor false boolean mentor = true, filter problems solved by mentor, applied only in codeforces, requires Authorization Token
hide_solved false boolean if hide_solved = true, problems solved by user will be excluded, requires Authorization Token
only_wrong false boolean if only_wrong = true, filter problems that is wrong attempt by user, requires Authorization Token
