Search for Userlists

This route helps in viewing all public lists with a suitable search parameter that matched with the name of a public list.

URL

GET
/lists/userlists/q={search_parameter}

Headers

Header Name Required Type Description
Authorization false string Bearer {Token}

URL Parameters

For GET Request. If URL params exist, specify them in accordance with name mentioned in URL section. Mention if is required or optional. Document data constraints.

Param Name Required Type Description
search_parameter true string Search parameter to find the public list

Status Code: 200
