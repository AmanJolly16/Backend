Success Code

List all the Success Code e.g. 200 STATUS OK etc.

HTTP Status Codes Purpose Definition
200 STATUS OK Standard response for successful HTTP requests. The actual response will depend on the request method used. In a GET request, the response will contain an entity corresponding to the requested resource. In a POST request the response will contain an entity describing or containing the result of the action.

Error Code

List all the error Code e.g. 404 NOT FOUND, 400 BAD REQUEST etc.

HTTP Status Codes Purpose Definition
400 BAD REQUEST The request cannot be fulfilled due to bad syntax. General error when fulfilling the request would cause an invalid state. Domain validation errors, missing data, etc. are some examples.
